Ghost Escape

Ghost Escape is a 2D Top-down Stealth Puzzle game made with Unity and C# using open-source sprites. You play (a rather adorable) ghost girl who one day finds her haunted home infiltrated by exterminators and weird humans. They've laid ghost-proof traps throughout the house, but you *possess* ghostly abilities they do not. Escape each room as you *haunt* for a new home.

UTS Discover

UTS Discover is an augmented reality (AR) experience for mobile devices that aims to promote the UTS Faculty of Engineering and IT (FEIT) to prospective international students. The application was built in the UTS Software Studio as a real, client-driven project for the FEIT, utilising agile development practises. The app, built in Unity, uses AR target-recognition to display video content and 3D models, as well as gyroscopic data to deliver a 360 image tour of UTS campus.

Memory Lane

Memory Lane is an interactive album, tablet application aimed to connect elderly users with younger members of their family by stimulating familiar experiences as users revive old memories through a nostalgic digital user experience. It aims to promote meaningful interactions through conversational text prompts throughout the journey of going through automatically-generated photo lanes. The prototype was developed using HTML, CSS and Javascript, utilising JQuery Libraries.

PlayBall

PlayBall is an interactive music program made using Processing. The user interaction is supported by Leap Motion to allow motion control of the hands with the program. Fingers control moveable green points that create sounds when they touch circles with the respected instrument name. Group Assignment for subject Interactive Media.
